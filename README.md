Pour utiliser ChocoScan :

Il est important de rappeler que l'utilisation de cet outil doit se faire à des fins éthiques et légales.

ChocoScan se lance de la manière suivante :

python chocoscan.py --scan [TARGET_IP] --export-html (pour avoir un rapport en html complet) :
